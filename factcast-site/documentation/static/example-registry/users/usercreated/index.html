UserCreated

Describes a user creation event.

Latest version (3)

Schema

{
  "additionalProperties" : true,
  "properties" : {
    "firstName" : {
      "type": "string"
    },
    "lastName" : {
      "type": "string"
    },
    "salutation": {
      "type": "string",
      "enum": ["Mr", "Mrs", "NA"]
    },
    "displayName": {
      "type": "string"
    }
  },
  "required": ["firstName", "lastName", "salutation", "displayName"]
}

Examples

simple.json

{
  "firstName" : "Horst",
  "lastName": "Lichter",
  "salutation": "Mr",
  "displayName": "Horst Lichter"
}
